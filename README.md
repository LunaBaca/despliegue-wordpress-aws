# Despliegue de WordPress en AWS EC2

Este proyecto detalla el proceso de implementación de un servidor web en la nube utilizando AWS, configurando un stack LAMP y desplegando WordPress.

## 🚀 Información del Proyecto
- **Estudiante:** Luna Baca
- **IP Pública:** http://34.228.39.22/index.php/nosotros/
- **Tecnologías:** AWS EC2, Ubuntu, Apache, MariaDB, PHP.

## 🛠️ Pasos Realizados
1. Creación de instancia EC2 y configuración de Security Groups.
2. Configuración de usuario personalizado `luna` con permisos sudo.
3. Instalación de Apache, MariaDB y PHP (Stack LAMP).
4. Creación de base de datos y usuario en MySQL.
5. Instalación y configuración de WordPress.
6. Aseguramiento del servidor con Firewall UFW.
